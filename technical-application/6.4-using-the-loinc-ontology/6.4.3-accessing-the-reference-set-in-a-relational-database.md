# 6.4.3 Accessing the Reference Set in a Relational Database

If the LOINC Ontology has been loaded in a relational database using a model that complies with the RF2 specification for tables and column names, the  content can be accessed using SQL queries in line with the examples below. To Set up SNOMED CT in a relational database, please follow the guidance and instructions provided in the [SQL Practical Guide](https://confluence.ihtsdotools.org/display/DOCSQLPG).
